Toto je projekt zamerany na visualizaciu algortmov na hladanie najkratsiej cesty, konkretne Djisktra a A* . Projekt je vytvoreny v Pythone, treba si doinstalovat kniznicu Pygame.

This projekt is about visualization path finding algorithms (Djisktra, A*) develop in pyhton. Instalation of Pygame library is needed.
